# 번역 관리

작업: $ARGUMENTS

---

## 파일 위치
```
i18n/locales/
├── en/{namespace}.json
└── ko/{namespace}.json
```

## 사용법
```tsx
import { useTranslations } from 'next-intl';
const t = useTranslations('namespace');
return <h1>{t('key')}</h1>;
```

## 규칙
1. en/ko 모두 업데이트
2. 중첩 객체로 섹션 구분: `section.key`
3. camelCase 사용

요청에 따라 번역 파일을 추가/수정해주세요.

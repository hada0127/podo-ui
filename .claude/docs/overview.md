# Podo UI 프로젝트 개요

## 소개
Podo UI는 SCSS 모듈 기반 디자인 시스템 + React UI 컴포넌트 라이브러리입니다.

## 핵심 특징
- **SCSS 우선**: 97% CSS 기반 (최대 유연성)
- **React 컴포넌트**: 3% (복잡한 상태 관리용)
- **테마 지원**: 라이트/다크 모드, 웜 톤
- **반응형**: PC(12) / Tablet(6) / Mobile(4) 그리드
- **아이콘 폰트**: SVG → WOFF 자동 생성

## 기술 스택
| 분류 | 기술 |
|------|------|
| 빌드 | Vite 5.4 |
| 프레임워크 | Vike 0.4 (vike-react) |
| 언어 | TypeScript |
| 스타일 | SCSS (CSS Modules) |
| 국제화 | i18next, react-i18next |
| 검증 | Zod |
| 배포 | Cloudflare Pages |
